# Builder

Planned first-party agentsdk builder app.

`docs/01_VISION.md` and `docs/03_ROADMAP.md`. The builder will dogfood agentsdk
resources, workflows, actions, tools, safety policy, and packaging to scaffold
agentic applications from guided requirements.
